## Run Locally

Clone the project

```bash
  git clone git@github.com:edderleonardo/BasicFastAPI.git
```

Go to the project directory

```bash
  cd booksAPI
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  uvicorn "main:app" --reload
```

Visit http://localhost:8000/docs
